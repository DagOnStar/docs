Getting starting¶

Repository¶

You can clone latest development version from our Git repository: https://github.com/DagOnStar/dagonstar.git

Prerequisites¶

  • Python 2.7

Installation using virtualenv¶

git clone https://github.com/DagOnStar/dagonstar.git
cd dagonstar
virtualenv venv
. venv/bin/activate
pip install -r requirements.txt
export PYTHONPATH=$PWD:$PYTHONPATH

Using it¶

This section describes how to use DagOn* to generate a simple data workflow.

  1. Create a configuration file in the working directory, you could use the configuration sample file (config.ini.sample)
cd dagonstar
mkdir working_dir
cp dagon.ini.sample working_dir/dagon.ini
cd working_dir
  1. Create a new Python file and import the next modules.
from dagon import Workflow
from dagon.task import DagonTask, Types
  1. Create the workflow orchestation object with the workflow name.
workflow = Workflow("DataFlow-Demo")
  1. Define three simple batch tasks, passing as a parameter the command to execute. workflow:///A/output/f1.txt means, that the file outpu/f1.txt have to be gotten from the scratch directory of the task A.
taskA = DagonTask(Types.BATCH, "A", "mkdir output;hostname > output/f1.txt")
taskB = DagonTask(Types.BATCH, "B", "echo $RANDOM > f2.txt; cat workflow:///A/output/f1.txt >> f2.txt")
taskC = DagonTask(Types.BATCH, "C", "echo $RANDOM > f2.txt; cat workflow:///A/output/f1.txt >> f2.txt")
taskD = DagonTask(Types.BATCH, "D", "cat workflow:///B/f2.txt >> f3.txt; cat workflow:///C/f2.txt >> f3.txt")
  1. Add the tasks to the workflow.
workflow.add_task(taskA)
workflow.add_task(taskB)
workflow.add_task(taskC)
workflow.add_task(taskD)
  1. Generate the dependencies between tasks and run the workflow.
workflow.make_dependencies()
workflow.run()
  1. Putting all together.
from dagon import Workflow
from dagon.task import DagonTask, Types

workflow = Workflow("DataFlow-Demo")

taskA = DagonTask(Types.BATCH, "A", "mkdir output;hostname > output/f1.txt")
taskB = DagonTask(Types.BATCH, "B", "echo $RANDOM > f2.txt; cat workflow:///A/output/f1.txt >> f2.txt")
taskC = DagonTask(Types.BATCH, "C", "echo $RANDOM > f2.txt; cat workflow:///A/output/f1.txt >> f2.txt")
taskD = DagonTask(Types.BATCH, "D", "cat workflow:///B/f2.txt >> f3.txt; cat workflow:///C/f2.txt >> f3.txt")

workflow.add_task(taskA)
workflow.add_task(taskB)
workflow.add_task(taskC)
workflow.add_task(taskD)

workflow.make_dependencies()
workflow.run()

You can find more examples with other kinds of tasks in Examples.
